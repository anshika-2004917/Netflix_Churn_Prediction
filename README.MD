# 🎬 Netflix Customer Churn Prediction

## 📌 Project Overview
This project is a Machine Learning model that predicts whether a Netflix customer will churn (leave) or stay based on their behavior and subscription details.

It helps identify customers at risk so that retention strategies can be applied.

## 🧠 Problem Statement
Customer churn is a major issue in subscription platforms like Netflix.  
This project classifies customers as:
- 1 → Churn ❌
- 0 → Not Churn ✅

## 📊 Features Used
- Age  
- Subscription Type  
- Watch Hours Per Week  
- Tenure (Months)  
- Monthly Charges  
- Payment Delays  
- Customer Satisfaction  
- Support Tickets  
- Genre Preference  

## ⚙️ Tech Stack
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Joblib  

## 🤖 Model Used
Random Forest Classifier
